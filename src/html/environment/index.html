キャリアパス
・当社ではゼネラリストとして入社後、営業・プランナー・データアナリストなどの業務を経験しながらキャリアを確立していただきます。
・一定の業務や地域において安定的に業務を遂行する職群を設けることで様々な働き方に対応しています。
・パフォーマンスによっては、入社後5年で管理者層に到達可能です。

管理者層

  • プロフェッショナル

    高い専門性を発揮しながらLMのビジネスをリードする。

  • マネジメント

    高い専門性を発揮しつつ、組織マネジメントを行う。

  • 最速5年目

  • ゼネラリスト

    複数の業務を経験しながらキャリアを形成する。

  • スペシャリスト

    一定の業務、地域において安定的に業務を遂行する。

評価制度
・当社は、働く上で大切にする価値基準として「コアバリュー」を定めています。
・当社の評価制度は、全社員共通の物指しとして「コアバリュー」に基づく評価、各職群の特性に応じた目標達成度評価による評価の2つから構成されています。
・管理者層の評価では業績のみならず職場環境や人材育成に関わる評価軸を持たせています。
・非管理職層の評価では、キャリア形成を意図した評価軸を持たせています。

バリュー評価

・「コアバリュー」から抽出した全社共通の価値観
・マインド

目標達成度評価

・業績目標
・管理者層では職場環境、人材育成、創造性など固有目標を付加
・非管理者層ではキャリアビジョンを基に目標を設定

バリュー処遇評価

・バリュー評価と業績・職群固有目標による評価の両面から昇格や賞与支給額を判断

人材育成
・当社の人材育成体系は新人育成から始まり、自己学習支援、基礎スキルや専門スキルの習得、全社員参加型のワークショップなどを実施しています。

管理者として求められる振る舞いを習得する機会を設けています。

専門スキル
（データ分析、マーケティングテクノロジー等）

当社で特に求められる専門スキルに関する育成施策を行っています。

全社育成
（「コアバリュー」ワークショップ等）

ビジネスベーシック
（ビジネス/分析、IT/Web、自己学習）

全社員共通で求められる研修や自己学習環境を整備しています。

異動・配置施策

自己申告や社内公募などの制度に加え、ローテーションや海外赴任などを実施しています。

新人育成（新人研修+OJT制度）

人材育成（新人OJT制度）
OJTトレーナーとトレーニーの共通言語となる若手育成クライテリアを設定し、若手OJT制度を組織的に運用しています。
  • 育成クライテリア：若手の間に培うべき、知識・スキル・経験・実績の設定
  • 育成方法：OJT、内部・外部研修、ローテーション
  • 配属先：営業、サービス企画・開発、データ分析、IT等の組織から選定
  • トレーナー：若手社員をインストラクターとして任命
  • 新人育成共有会：OJTペアによる共有会を運営し、かつ全社的に新人の成長する姿を共有
休日・休暇制度
・当社ではライフイベントに合わせた休暇制度を整備しています。 ・また、連続休暇を奨励するなど年次有給休暇取得の促進につとめています。
内容
慶弔休暇（結婚、配偶者出産、忌引き） 本人結婚：5日、配偶者出産：2日、
父母・配偶者・子の死亡：5日、
祖父母・配偶者の父母・兄弟・姉妹の死亡：3日
産前・産後休暇 産前：6週間（多胎妊娠は、14週間）
産後：8週間
※給与は健保より支給されます
母性健康管理休暇 妊娠・出産後1年以内の方については、法律に基づき通院することができます。
妊娠23週：4週に1回、
妊娠24週から35週まで：2週に1回、
妊娠36週から出産まで：1週に1回、
産後1年以内：医師等の指示による回数
生理休暇 就業が著しく困難な場合、一日単位で休暇請求が可能（給与は支給しない）。
看護休暇 小学校就学前の子一人につき年5日、2人以上は年10日。
裁判員休暇 裁判所への出頭、裁判審理に参加する場合において必要な範囲内で休暇を支給。
連続休暇奨励 年次有給休暇日数のうち、最低年間５日については、時季を指定して、連続取得することができる。
